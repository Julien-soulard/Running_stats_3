# Running Stats

Visualisation de mes statistiques de running.

## Comment mettre à jour les données

1. Éditez le fichier `data.csv`
2. Ajoutez une nouvelle ligne avec le format : `date,semaine,seance,duree_min,distance_km`
3. Committez et pushez les changements

## Développement local

Ouvrez simplement `index.html` dans votre navigateur.
